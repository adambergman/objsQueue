objsQueue - Native Twitter Example

This example will queue an objective-c function to send a Tweet using the iOS5 built in Twitter API. Note that this must be running on an actual device with a Twitter account setup. If you have not added a Twitter account in the Settings of iOS or if you are running on the simulator this call will appear to do nothing.

function sendTweet()
{
    var didComplete = function(obj){};
    var didFail = function(obj){ alert('Command Failed!'); };
    objsQueue.add("sendTweet:", {params:['Tweet Tweet!'], complete:didComplete, fail:didFail});
}

function openTweet()
{
    objsQueue.add("openTweet", {fail:function(obj){ alert('Command Failed!'); }});
}
